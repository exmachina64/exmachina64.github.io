CSS Selectors

Basic Selectors

Selector Syntax Description Examples
Universal Selector * Selects all elements. * { style properties }
Type Selector element Selects all elements that have the given node name. span { style properties }
Class Selector .class_name Selects all elements that have the given class attribute. .fancy { style properties }
ID Selector #id_value Selects an element based on the value of its id attribute. #identified { style properties }
Attribute Selector a[title] Selects all elements that have the given attribute. a { style properties }

Group Selectors

Selector Syntax Description Examples
Selector List element, element, element The , selector is a grouping method that selects all matching nodes. h1, h2, h3, h4, h5, h6 { style properties }

Combinators

Selector Syntax Description Examples
Descendant Combinator selector1 selector2 Combines two selectors such that elements matched by the second selector are selected if they have an ancestor element matching the first selector. li li { style properties }
Child Combinator selector1 > selector2 Matches only those elements matched by the second selector that are the direct children of elements matched by the first. div > span { style properties }
General Sibling Combinator former_element ~ target_element Separates two selectors and matches all iterations of the second element that are following the first element, though not necessarily immediately, and are children of the same parent. p ~ span { style properties }
Adjacent Sibling Combinator former_element + target_element Separates two selectors and matches the second element only if it immediately follows the first element and both are children of the same parent element. li:first-of-type + li { style properties }
Column Combinator column-selector||cell-selector Matches only those elements matched by the second selector that belong to the column elements matched by the first. col.selected||td { style properties }

Pseudo-Classes and Pseudo-Elements

Selector Syntax Description Examples
Pseudo-Classes selector:pseudo-class A keyword added to a selector that specifies a special state of the selected element(s). button:hover { style properties }
Pseudo-Elements selector::pseudo-element A keyword added to a selector that lets you style a specific part of the selected element(s). p::first-line { style properties }
